# Deployment Checklist

## Before You Start
- [ ] Code is working locally
- [ ] All forms tested
- [ ] WhatsApp button working
- [ ] Images loading

## GitHub Setup
- [ ] GitHub account created
- [ ] GitHub Desktop installed
- [ ] Repository created
- [ ] Code uploaded
- [ ] Repository published

## Vercel Deployment
- [ ] Vercel account created (with GitHub)
- [ ] Project imported
- [ ] Build settings configured
- [ ] First deployment successful
- [ ] Website URL received

## Backend Deployment (Railway)
- [ ] Railway account created
- [ ] Project created
- [ ] Backend deployed
- [ ] MongoDB connected (Atlas or Railway)
- [ ] Environment variables added:
  - [ ] MONGO_URL
  - [ ] DB_NAME
  - [ ] SMTP_SERVER
  - [ ] SMTP_PORT
  - [ ] SMTP_USERNAME
  - [ ] SMTP_PASSWORD
- [ ] Domain generated
- [ ] Backend URL copied

## Connect Frontend & Backend
- [ ] Added REACT_APP_BACKEND_URL to Vercel
- [ ] Redeployed frontend
- [ ] Tested API connection

## Email Setup
- [ ] Gmail 2FA enabled
- [ ] App password generated
- [ ] SMTP credentials added to Railway
- [ ] Test email sent successfully

## Final Testing
- [ ] Open website URL
- [ ] Test WhatsApp button
- [ ] Fill enquiry form
- [ ] Verify email received
- [ ] Test on mobile
- [ ] Test on different browsers
- [ ] Check page load speed

## Optional
- [ ] Custom domain purchased
- [ ] DNS configured
- [ ] SSL certificate active
- [ ] Analytics added (Google Analytics)

## Costs
- [ ] Understood free tier limits
- [ ] Ready for $5/month Railway cost
- [ ] Domain renewal reminder set

## Backup
- [ ] Code backed up on GitHub
- [ ] Environment variables documented
- [ ] Database backup enabled
